None blocking release. Two notes:

We frame this as feature not bug: "study the canonical agent loop, then graduate to AutoGPT Platform / LangGraph / Anthropic Agents SDK." Multi-model support across 8 providers (Phase G) means students can run the curriculum end-to-end on cheap modern models (DeepSeek, Qwen, Groq, local llama-3.3) without an OpenAI bill.

The s03 doc dedicates a section to the side-by-side translation, and `provider_openai.go` does the wire conversion.
